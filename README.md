[Butterscotch](https://github.com/MrPowerGamerBR/Butterscotch) is an open-source
GameMaker: Studio runner implementation designed to run across a wide variety of
embedded and console platforms.

This is a fork of Butterscotch specifically targeting the PlayStation 2.

Thanks to MrPowerGamerBR, Fancy2209, and all of the contributors to upstream
Butterscotch.

## Game Compatibility

This fork specifically targets Undertale v1.08 (GameMaker: Studio 1.4.1804, Bytecode Version 16)
with the goal of making it stable, fully playable and optimized without performance issues.

Other GMS games, such as Deltarune, may be targeted in the future.

**NOTE**: Games other than Undertale have not been tested and are not currently supported.
